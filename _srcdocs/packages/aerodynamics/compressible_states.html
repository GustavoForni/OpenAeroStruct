Class definition for CompressibleVLMStates.

class openaerostruct.aerodynamics.compressible_states.CompressibleVLMStates(**kwargs)[source]¶

Bases: openmdao.core.group.Group

Group that contains the states for a compressible aerodynamic analysis.

This is done in three steps:

  1. Convert all VLM geometries to Prandtl-Glaert domain.

2. Solve the VLM problem in the Prandtl-Glaert domain using VLMStates group.

3. Convert the resulting forces back to the physical domain through the inverse Prandtl-Glauert transform to recover the compressible forces.
